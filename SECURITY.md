# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by email:

- `info@matejzavadil.cz`

Please do not open a public issue for security reports.

## What to Include

To help us investigate, include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact
- Affected versions

## Response Timeline

We aim to acknowledge reports within 3 business days and provide a status update within 10 business days.
